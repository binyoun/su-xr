# 宿 · Su, Where the Moon Rests

**Su** (宿) is an XR installation work by Bin Youn exploring the ancient Chinese lunar mansion system (二十八宿) as a framework for spatial, celestial, and embodied experience.

> *One Night the Moon Stayed. That Was Yours.*

---

## Overview

The 28 lunar mansions (二十八宿) divide the sky into stations — each a resting place for the moon on its monthly journey. **Su** uses this cosmological map as the architecture for an immersive installation, connecting personal birthdate to celestial palace, element, and tone.

This work is part of the **Gelora XR** ecosystem → [gelora-xr.xyz](https://gelora-xr.xyz)

**Exhibition:** Clouds upon the Tree · Quán Nghệ · HCMC 2026
**Artist:** Bin Youn · RMIT University Vietnam

## Tools

### `tools/mansion-finder.html`
**二十八宿 · Lunar Mansion Finder**

A dark-themed browser tool. Enter a birthdate → receive your celestial mansion and palace.

- Calculates mansion index via Julian Day Number anchored to Jan 1, 2000 (虛 · The Void)
- Four palaces: 青龍 Green Dragon (East) · 朱雀 Vermilion Bird (South) · 白虎 White Tiger (West) · 玄武 Black Tortoise (North)
- Each mansion returns: Chinese character, pinyin, Korean reading, meaning, associated star, poetic line, element, planet, pentatonic tone
- Connects to the Gelora XR experience via `gelora-xr.xyz/direction.html`
